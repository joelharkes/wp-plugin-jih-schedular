A schedule in your wordpress website where (unregistered) people can sign up for an hour.

== Description ==

A new plugin where people can schedule in hours into a timetable.
easily when they are registered or with recaptchawhen they are not.

This plugin is heavenly reliant on you using a theme which uses bootstrap ass layout base.

== Installation ==

Install the plugin. A menu will be added to the admin menu.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress and you're done installing

=== Configuration ===

1. Go to this menu and add the reCaptcha keys in the settings.
2. Add the calendars you want.
3. Add the post, added by this plugin, in your menu so people can go to this page.
